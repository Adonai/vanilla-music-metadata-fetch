This app is a plugin. Prior to use it you should have
<a href="https://play.google.com/store/apps/details?id=ch.blinkenlights.android.vanilla">Vanilla Music</a> installed. Once both player and this plugin
are installed, request a context menu from library screen, select "Plugins" and
"Metadata fetch" afterwards. Note that this plugin requires internet connection to
be useful.

Features:

* Fetches tag metadata for various music files (mp3/ogg/flac)
* Can also fetch cover image for selected song
* Can write retrieved cover and tags if <a href="https://play.google.com/store/apps/details?id=com.kanedias.vanilla.audiotag">Tag Plugin</a> is installed

This app uses <a href="https://acoustid.org">AcoustID</a> as a service to match fingerprints. It is the only service that
is built upon libre software and doesn't require payment to use. Fetches following from AcoustID:

    * Title
    * Artist
    * Album
    * Album Artist
    * Release country
    * Year of first release
    * Track number/track count
